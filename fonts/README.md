# Fonts Directory

This directory should contain the Loew Next Arabic font files.

## Required Font Files

Place the following font files in this directory:

- `LoewNextArabic-Regular.woff2` (or `.woff` / `.ttf`)
- `LoewNextArabic-Medium.woff2` (or `.woff` / `.ttf`)
- `LoewNextArabic-Bold.woff2` (or `.woff` / `.ttf`)
- `LoewNextArabic-Light.woff2` (or `.woff` / `.ttf`)

## Font Weights Used

- **300** (Light) - for light text
- **400** (Regular) - for normal text
- **500** (Medium) - for medium weight text
- **700** (Bold) - for bold text

## Note

The CSS is configured to use these font files. If you have different file formats or names, update the `@font-face` declarations in `styles.css` accordingly.

## Font Source

Loew Next Arabic is a commercial font. Ensure you have the proper license to use it in your project.
